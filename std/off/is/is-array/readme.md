# isArray


## DESCRIPTION

`isArray` - Determine if value is an array.


**SYNOPSIS:**    
`isArray(value)`

*PARAMETERS:*    
- `value` Any value.

**RETURN VALUE:**    
Boolean: `true` if value is an array, `false` otherwise.


## INSTALL

[npm](https://www.npmjs.com/)

```shell
$ npm i @ivanilic/is-array
```


# USAGE

```js
const isArray = require('@ivanilic/is-array');

isArray([]);
//=> true
```
