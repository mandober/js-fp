# Lambda Calculus Interpreter

The most basic interpreter for λ-calculus in JS.

Dir: `/src/lc/`, `/src/lc/lang/`

Files
- `token.js`
- `lexer.js`
- `parser.js`
- `ast.js`
- `interpreter.js`
- `lc.js` Main file.
- `lambdas.lc` File with lambda expression

Synopsis: `node lc [--ast] <file>`
